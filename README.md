# agent-plugins

Personal plugin marketplace for Claude Code and Claude Cowork.

## Structure

- **`/plugins`** — plugins developed and maintained here

## Installation

```
/plugin install ship-check@agent-plugins
```

Or browse via `/plugin > Discover`.

## Plugins

| Plugin | Description |
|--------|-------------|
| [ship-check](plugins/ship-check/) | Fresh-eyes review agents for the ship-check pipeline (pr-reviewer, code-quality-reviewer, test-auditor, bug-checker) |
